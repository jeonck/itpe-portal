# 정의
소프트웨어가 명시된 요구사항을 만족하고 사용자 기대를 충족하는 정도를 나타내는 특성으로, ISO/IEC 25010 품질 모델로 정의됩니다.

## 특징
- ISO/IEC 25010 주특성: 기능 적합성, 성능 효율성, 호환성, 사용성, 신뢰성, 보안, 유지보수성, 이식성
- ISO 9126 (구버전): 기능성, 신뢰성, 사용성, 효율성, 유지보수성, 이식성
- 각 주특성은 여러 부특성으로 세분화
- 품질 메트릭을 통한 정량적 측정
- 품질 보증(QA)과 품질 관리(QC) 활동
