# 정의
IT 거버넌스는 조직의 목표 달성을 위해 IT 자원과 프로세스를 효과적으로 통제·관리하는 체계로, IT 투자의 가치를 극대화하고 리스크를 최소화하는 의사결정 구조입니다.

## 특징
- **ISO 38500**: IT 거버넌스 국제 표준으로 6대 원칙(책임성, 전략, 인수, 성과, 적합성, 인간행동)과 3대 활동(평가-지시-모니터링) 제시
- **COBIT(Control Objectives for Information and related Technology)**: ISACA에서 제정한 IT 거버넌스 프레임워크로, 5대 원칙(이해관계자 요구 충족, 전사 통합, 단일 프레임워크 적용, 전체적 접근, 거버넌스와 관리 분리)과 EDM(평가-지시-모니터링) 프로세스
- **IT-BSC(Balanced ScoreCard)**: IT 성과를 4가지 관점(재무, 고객, 내부프로세스, 학습/성장)에서 균형있게 측정하는 성과관리 도구
- **핵심 영역**: IT 전략 정렬, IT 가치 제공, 리스크 관리, 자원 관리, 성과 측정
