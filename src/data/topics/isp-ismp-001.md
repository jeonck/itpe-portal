# 정의
ISP(Information Strategy Planning)는 조직의 중장기 정보화 전략을 수립하는 방법론이며, ISMP(Information System Master Plan)는 정보시스템 구축을 위한 마스터플랜으로, 조직의 비전과 목표를 IT로 구현하기 위한 청사진입니다.

## 특징
- 수립 절차: 1) 현행 시스템 분석 (As-Is) → 2) 미래 모델 설계 (To-Be) → 3) 이행 계획 수립 (Gap Analysis) → 4) 실행 로드맵 작성
- 주요 산출물: 현행 분석서, 요구사항 정의서, 목표 아키텍처, 응용 시스템 구성도, 데이터 아키텍처, 기술 아키텍처, 이행 계획서, 투자 계획서
- ISP vs ISMP 차이점: ISP는 전략 중심(What, Why)으로 3-5년 계획, ISMP는 실행 중심(How)으로 1-3년 구체적 실행계획
- 핵심 활동: SWOT 분석, CSF(핵심성공요인) 도출, BSC(균형성과지표) 설정, EA 프레임워크 적용
