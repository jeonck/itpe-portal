# 정의
데이터베이스의 중복을 최소화하고 데이터 무결성을 향상시키기 위해 테이블을 분해하는 과정입니다.

## 특징
- 데이터 중복 최소화
- 이상 현상 방지 (삽입/삭제/갱신 이상)
- 데이터 무결성 향상
- 저장 공간 효율화
