# 정의
EVM(Earned Value Management, 획득가치 관리)은 프로젝트의 범위, 일정, 원가 성과를 통합적으로 측정하고 모니터링하여 프로젝트의 현재 상태와 미래 예측을 제공하는 핵심 관리 기법입니다. 이를 통해 프로젝트 관리자는 현재까지의 성과를 객관적으로 파악하고, 필요한 경우 시정 조치를 취하여 프로젝트를 성공적으로 완료할 수 있도록 합니다.

## 특징
- **3대 기본 요소**:
    - **PV (Planned Value, 계획가치)**: 특정 시점까지 계획된 작업에 투입될 예산. (BPWS: Budgeted Cost of Work Scheduled)
    - **EV (Earned Value, 획득가치)**: 특정 시점까지 실제로 완료된 작업에 대한 계획 예산. (BCWP: Budgeted Cost of Work Performed)
    - **AC (Actual Cost, 실제원가)**: 특정 시점까지 실제로 완료된 작업에 투입된 실제 비용. (ACWP: Actual Cost of Work Performed)
- **주요 성과 지표**:
    - **SV (Schedule Variance, 일정 편차)**: `EV - PV`
        - SV > 0: 계획보다 일정이 빠름
        - SV < 0: 계획보다 일정이 지연됨
    - **CV (Cost Variance, 원가 편차)**: `EV - AC`
        - CV > 0: 계획보다 원가 절감 (예산 내)
        - CV < 0: 계획보다 원가 초과 (예산 초과)
    - **SPI (Schedule Performance Index, 일정 성과 지수)**: `EV / PV`
        - SPI > 1: 계획보다 일정이 빠름 (효율적)
        - SPI < 1: 계획보다 일정이 지연됨 (비효율적)
    - **CPI (Cost Performance Index, 원가 성과 지수)**: `EV / AC`
        - CPI > 1: 계획보다 원가 절감 (효율적)
        - CPI < 1: 계획보다 원가 초과 (비효율적)
- **주요 예측 지표**:
    - **EAC (Estimate At Completion, 완료 시점 예상 원가)**: `AC + (BAC - EV) / CPI` (CPI로 미래 성과 예측) 또는 `BAC / CPI`
    - **ETC (Estimate To Complete, 잔여 작업 예상 원가)**: `EAC - AC`
    - **BAC (Budget At Completion, 전체 예산)**: 프로젝트의 총 예산.
