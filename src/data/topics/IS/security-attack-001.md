# 정의
정보 시스템의 기밀성, 무결성, 가용성을 침해하기 위한 다양한 사이버 공격 기법들입니다.

## 특징
- DDoS (Distributed Denial of Service): 분산 서비스 거부 공격, 가용성 침해
- Ransomware: 데이터 암호화 후 금전 요구
- APT (Advanced Persistent Threat): 지능형 지속 위협, 장기간 잠복
- SQL Injection: DB 쿼리 조작을 통한 정보 탈취
- XSS (Cross-Site Scripting): 악성 스크립트 삽입, CSRF (요청 위조)

## 최신 트렌드
- AI 기반 공격
- Supply Chain Attack
- Zero-day Exploit
