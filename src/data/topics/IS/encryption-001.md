# 정의
평문을 암호문으로 변환하여 정보의 기밀성을 보장하는 기술입니다.

## 특징
- 대칭키 암호화: 동일한 키로 암호화/복호화 (빠름)
- 비대칭키 암호화: 공개키/개인키 쌍 사용 (안전)
- 해시 함수: 단방향 암호화
- 디지털 서명: 무결성과 인증 보장

## 최신 트렌드
- 양자 암호
- 동형 암호
- Post-Quantum Cryptography
