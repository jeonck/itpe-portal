# 정의
망분리는 외부 인터넷망과 업무망을 분리하여 외부로부터의 위협이 내부 업무망으로 침투하는 것을 원천적으로 차단하는 보안 기술입니다. 망연계는 분리된 망(인터넷망과 업무망) 간에 필요한 데이터만을 안전하게 전송할 수 있도록 하는 기술입니다. 이 두 가지는 특히 금융, 공공 등 높은 수준의 보안이 요구되는 기관에서 내부 정보 유출 및 외부 공격 방어를 위한 핵심적인 보안 인프라로 활용됩니다.

## 특징
- 망분리 (Network Separation):
    - 목표: 외부 인터넷망을 통한 악성코드 유입 및 내부 정보 유출 방지.
    - 유형:
        - 물리적 망분리: 인터넷용 PC와 업무용 PC를 각각 사용하여 물리적으로 망을 분리하는 방식. 가장 높은 보안성을 제공하지만, 구축 및 관리 비용이 높고 사용자 불편이 큽니다.
        - 논리적 망분리: 하나의 PC에서 가상화 기술(VDI, SBC 등)을 이용하여 인터넷 환경과 업무 환경을 논리적으로 분리하는 방식. 물리적 망분리 대비 비용 효율적이고 사용자 편의성이 높습니다.
            - VDI (Virtual Desktop Infrastructure): 개인별 가상 PC 환경을 중앙 서버에서 제공.
            - SBC (Server-Based Computing): 서버의 컴퓨팅 자원을 공유하여 다수 사용자에게 가상 환경 제공.
    - 장점: 외부 공격으로부터 내부 업무망을 원천 차단하여 높은 보안성 확보, 내부 정보 유출 위험 감소.
    - 단점: 구축 비용 및 관리 복잡성 증가, 사용자 불편 초래, 망간 자료 전송의 어려움.
- 망연계 (Network Interconnection):
    - 목표: 분리된 망(인터넷망, 업무망, 대외망 등) 간에 필요한 데이터만을 안전하고 통제된 방식으로 전송.
    - 기능:
        - 단방향/양방향 전송: 데이터 전송 방향 제어.
        - 데이터 필터링: 보안 정책에 따라 전송될 데이터를 검증하고 필터링.
        - 악성코드 검사: 전송되는 모든 파일 및 데이터에 대한 악성코드 검사.
        - 전송 승인: 망간 자료 전송 시 관리자 승인 절차.
    - 주요 기술: 망연계 솔루션(보안 게이트웨이), 망간 자료 전송 시스템.
- 관계: 망분리를 통해 네트워크를 분리한 후, 망연계 기술을 통해 분리된 망 간의 필수적인 통신 및 자료 전송을 안전하게 허용하여 업무 효율성과 보안성을 동시에 확보합니다.
