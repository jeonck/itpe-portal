# 정의
사회공학적 해킹(Social Engineering)은 인간의 심리적 취약점이나 신뢰를 이용하여 개인이나 조직의 정보 시스템에 대한 접근 권한을 얻거나 기밀 정보를 탈취하는 비기술적인 공격 기법입니다. 대표적인 유형으로는 피싱(Phishing)과 스미싱(Smishing)이 있으며, 이는 단순히 기술적 취약점을 넘어 인간의 실수를 유도하여 보안을 침해합니다.

## 특징
- 목표: 사용자 심리를 이용하여 비밀번호, 개인 정보, 금융 정보 등을 탈취하거나 악성코드를 설치.
- 주요 원리:
    - 신뢰: 공격자가 신뢰할 수 있는 사람이나 기관을 가장하여 접근.
    - 긴급성: 즉각적인 조치를 요구하여 피해자가 충분히 생각할 시간을 주지 않음.
    - 호기심/두려움: 사용자의 호기심을 자극하거나 두려움을 유발하여 특정 행동을 유도.
- 주요 유형:
    - 피싱 (Phishing):
        - 정의: 특정 기관(금융기관, 공공기관 등)을 사칭한 이메일이나 웹사이트를 통해 개인 정보, 금융 정보 등을 탈취하는 공격.
        - 스피어 피싱 (Spear Phishing): 특정 개인이나 조직을 목표로 한 맞춤형 피싱. (CEO 사기, BEC 공격)
        - 작살 낚시 (Whaling): 고위 임원(CEO, CFO)을 대상으로 한 스피어 피싱.
        - 클론 피싱 (Clone Phishing): 정상적인 이메일을 위조하여 악성 링크나 첨부 파일을 포함시킨 이메일.
    - 스미싱 (Smishing):
        - 정의: SMS(문자 메시지)를 이용한 피싱과 악성코드의 합성어. 문자 메시지 내 악성 URL 클릭을 유도하여 악성 앱 설치나 개인 정보 탈취.
        - 주요 사례: 택배 사칭, 청첩장, 지인 사칭, 공공기관 사칭.
    - 파밍 (Pharming): 사용자가 정상적인 웹사이트 주소를 입력해도 가짜 웹사이트로 접속하도록 유도하여 개인 정보 탈취. (DNS 변조, 호스트 파일 변조)
    - 보이스 피싱 (Voice Phishing): 전화를 통해 개인 정보나 금융 정보를 요구하거나, 악성 앱 설치를 유도하는 공격. (검찰, 금융감독원 사칭)
    - 큐싱 (Qshing): QR 코드를 이용한 피싱. 악성 QR 코드 스캔 시 악성 웹사이트로 연결되거나 앱 다운로드 유도.
- 대응 방안:
    - 출처 불분명 이메일/문자 메시지 경계.
    - URL 및 첨부 파일 확인 습관화.
    - 금융 정보 및 개인 정보 입력 요구 시 공식 채널 확인.
    - 최신 보안 소프트웨어 사용 및 업데이트.
    - 다중 인증(MFA) 활성화.
