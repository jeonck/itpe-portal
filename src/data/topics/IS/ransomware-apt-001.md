# 정의
랜섬웨어(Ransomware)는 사용자 또는 조직의 시스템 접근을 제한하거나 데이터를 암호화한 후, 이를 해제하는 대가로 금전(랜섬)을 요구하는 악성코드 공격입니다. APT(Advanced Persistent Threat, 지능형 지속 위협) 공격은 특정 목표를 설정하여 다양한 공격 기법과 제로데이 취약점 등을 활용, 장기간에 걸쳐 은밀하게 정보를 탈취하거나 시스템을 파괴하는 고도화된 공격입니다.

## 특징
- 랜섬웨어 (Ransomware):
    - 공격 방식:
        - 암호화: 파일 또는 시스템을 암호화하여 접근 불가하게 만듦.
        - 화면 잠금: 시스템 화면을 잠가 사용을 방해.
    - 주요 유형:
        - CryptoLocker: 파일 암호화 후 비트코인 요구.
        - WannaCry: SMB 취약점을 이용한 전파.
    - 최신 동향:
        - 다중 협박 (Double Extortion): 데이터를 암호화하는 동시에 유출하여 공개하겠다고 협박.
        - RaaS (Ransomware-as-a-Service): 랜섬웨어 서비스를 판매하여 공격자가 쉽게 랜섬웨어 공격을 수행하도록 지원.
    - 대응: 백업, 최신 보안 패치, 의심스러운 파일/링크 열람 금지, 보안 솔루션 강화.
- APT 공격 (Advanced Persistent Threat):
    - 목표: 특정 조직이나 기업의 중요 정보 탈취, 시스템 파괴 등.
    - 공격 단계:
        1.  초기 침투: 스피어 피싱, 제로데이 취약점 활용.
        2.  거점 확보: 시스템에 백도어 설치.
        3.  권한 상승: 시스템 내에서 더 높은 권한 획득.
        4.  내부 정찰: 네트워크 내부 정보 수집.
        5.  측면 이동: 다른 시스템으로 확산.
        6.  데이터 수집 및 유출: 목표 데이터 탈취.
        7.  흔적 삭제: 공격 흔적 제거.
    - 특징:
        - 지능적: 다양한 공격 기법을 조합.
        - 지속적: 장기간에 걸쳐 은밀하게 공격을 지속.
        - 특정 목표: 명확한 목표를 설정.
        - 회피성: 탐지 시스템을 우회.
    - 대응: 지능형 지속 위협 방어(APT Solution), 보안 관제 강화, 위협 인텔리전스 활용, 사용자 보안 교육.
