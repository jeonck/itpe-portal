# 정의
통계 분석은 데이터를 수집·정리·해석하여 패턴을 발견하고 예측하는 과학적 방법으로, 의사결정의 객관적 근거를 제공합니다.

## 특징
- 회귀분석: 독립변수(X)와 종속변수(Y) 간 관계를 모델링. 선형회귀(y=ax+b), 다중회귀(y=b0+b1x1+b2x2+...), 로지스틱 회귀(분류). R²(결정계수)로 모델 설명력 평가
- 가설검정과 P-value: 귀무가설(H0)과 대립가설(H1) 설정 후 통계량 계산. P-value는 귀무가설이 참일 때 관측된 결과가 나올 확률로, P<0.05면 통계적으로 유의하다고 판단(귀무가설 기각). t-검정, 카이제곱 검정, ANOVA 등
- 상관분석: 두 변수 간 선형 관계 강도를 측정. 피어슨 상관계수(r)는 -1~1 범위로 1에 가까우면 강한 양의 상관, -1에 가까우면 강한 음의 상관. 상관관계 ≠ 인과관계 주의
- 기타 기법: 기술통계(평균, 중앙값, 표준편차), 확률분포(정규분포, 이항분포), 신뢰구간, 표본추출
