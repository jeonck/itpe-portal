# 정의
자연 선택과 유전 법칙을 모방하여 최적해를 찾는 진화 기반 최적화 알고리즘입니다.

## 특징
- 생물학적 영감: 선택, 교배, 돌연변이
- 구성 요소: 유전자(Gene), 염색체(Chromosome), 개체(Individual), 세대(Generation)
- 과정: 1) 초기 집단 생성 → 2) 적합도 평가 → 3) 선택 → 4) 교배(Crossover) → 5) 돌연변이(Mutation) → 반복
- 선택 (Selection): 룰렛 휠, 토너먼트, 순위 기반
- 교배 (Crossover): 1점/2점/균등 교배, 부모 유전자 조합
- 돌연변이 (Mutation): 랜덤 변이, 지역 최적 탈출
- Fitness Function: 해의 적합도 평가, 문제 의존적
- 장점: 복잡한 최적화, 전역 탐색, 단점: 느림, 수렴 보장 없음
- 활용: 일정 최적화, 경로 최적화, 게임 AI, 신경망 구조 탐색
