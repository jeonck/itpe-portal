# 정의
블랙박스 AI 모델의 예측 과정을 설명하고 해석 가능하게 만드는 설명 가능한 AI 기술.

## 특징
- 필요성: 신뢰성, 책임성, 규제 준수 (GDPR), 디버깅
- Model-agnostic: 모델 종류와 무관하게 적용 가능
- LIME (Local Interpretable Model-agnostic Explanations): 지역적 설명
- 동작: 예측 주변 데이터 샘플링 → 단순 모델 학습 → 해석
- SHAP (SHapley Additive exPlanations): Shapley Value (게임 이론)
- 각 특징의 기여도 계산, 공정한 분배, 전역+지역 설명
- Feature Importance: 특징 중요도, 트리 기반 모델
- Attention Visualization: Transformer, CNN의 주의 집중 영역
- Saliency Map: 이미지에서 중요 영역 시각화
- Grad-CAM: CNN 예측 근거 시각화
- 활용: 의료 진단, 금융 대출, 자율주행, 법률
