# 정의
네트워크에서 최적의 경로를 찾아 패킷을 전달하기 위한 프로토콜로, 내부/외부 라우팅 프로토콜로 구분됩니다.

## 특징
- 거리 벡터 (Distance Vector): RIP, 홉 수 기반, Bellman-Ford 알고리즘
- 링크 상태 (Link State): OSPF, 전체 토폴로지 인식, Dijkstra 알고리즘
- BGP (Border Gateway Protocol): AS 간 라우팅, Path Vector 방식
- OSPF: Area 개념, SPF 트리, Fast Convergence
- 라우팅 메트릭: 홉 수, 대역폭, 지연, 비용
