# 정의
공용 네트워크를 통해 암호화된 터널을 생성하여 안전한 사설 네트워크 연결을 제공하는 기술입니다.

## 특징
- 목적: 원격 접속, 데이터 암호화, IP 주소 우회
- 프로토콜: IPsec, SSL/TLS VPN, PPTP, L2TP, WireGuard
- IPsec: 네트워크 계층 암호화, Site-to-Site VPN
- SSL VPN: 웹 브라우저 기반, 클라이언트 간편
- WireGuard: 최신 프로토콜, 단순하고 빠름, 강력한 암호화
- 터널링: 데이터를 암호화하여 캡슐화
- Split Tunneling: VPN 트래픽과 일반 트래픽 분리
- 사용 사례: 원격 근무, 본사-지사 연결, 보안 접속

## 최신 트렌드
- Zero Trust Network Access (ZTNA)
- WireGuard 확산
