# 정의
저전력, 광역 통신을 목표로 하는 IoT 전용 네트워크 기술로, LoRa, NB-IoT, Zigbee 등이 대표적입니다.

## 특징
- LPWAN (Low Power Wide Area Network): 저전력, 장거리, 저속
- LoRa: 비면허 대역 (ISM), 장거리 (수 km~15km), 저전력
- LoRaWAN: LoRa 기반 프로토콜, Class A/B/C, 게이트웨이 방식
- NB-IoT: 3GPP 표준, 면허 대역, 통신사 인프라 활용
- Zigbee: 저전력, 짧은 거리 (10~100m), 메쉬 네트워크
- 사용 사례: 스마트 미터, 자산 추적, 농업 센서, 스마트 시티
- Matter: IoT 통합 표준 (Apple, Google, Amazon 등 주도)
- 비교: WiFi (고속, 근거리), Bluetooth (저전력, 근거리), Cellular (광역, 고비용)
