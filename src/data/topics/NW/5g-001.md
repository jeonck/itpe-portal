# 정의
초고속, 초저지연, 초연결을 목표로 하는 5세대 이동통신으로, eMBB, URLLC, mMTC 3대 서비스로 구성됩니다.

## 특징
- eMBB (Enhanced Mobile Broadband): 초고속 데이터 (최대 20Gbps)
- URLLC (Ultra-Reliable Low-Latency Communication): 초저지연 (1ms 이하)
- mMTC (Massive Machine Type Communication): 초연결 (100만 기기/km²)
- 주파수: Sub-6GHz (커버리지), mmWave (28/39GHz, 속도)
- Network Slicing: 용도별 가상 네트워크 분리
- MEC (Multi-access Edge Computing): 엣지 컴퓨팅 연동
- NSA vs SA: Non-Standalone (4G 코어), Standalone (5G 코어)
- 사용 사례: 자율주행, 원격 수술, 스마트 팩토리, AR/VR
