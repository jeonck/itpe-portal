# 정의
지리적으로 분산된 서버 네트워크를 통해 콘텐츠를 사용자와 가까운 위치에서 제공하여 속도와 가용성을 향상시키는 서비스입니다.

## 특징
- 동작 원리: 엣지 서버에 콘텐츠 캐싱, 가장 가까운 서버에서 제공
- 장점: 지연 시간 감소, 대역폭 절감, 가용성 향상, DDoS 방어
- 캐싱 전략: TTL (Time to Live), Cache-Control 헤더
- 주요 업체: Cloudflare, AWS CloudFront, Akamai, Fastly
- PoP (Point of Presence): CDN 엣지 서버 위치
- Origin Server: 원본 콘텐츠 서버
- Purge/Invalidation: 캐시 무효화
- 사용 사례: 동영상 스트리밍, 정적 파일 배포, API 가속
