# 정의
네트워크를 소프트웨어로 제어하는 SDN과, 네트워크 기능을 가상화하는 NFV를 통해 유연하고 자동화된 네트워크를 구축합니다.

## 특징
- SDN: 제어 플레인(Control Plane)과 데이터 플레인(Data Plane) 분리
- OpenFlow: SDN 프로토콜, 스위치를 프로그래밍 가능하게 제어
- SDN 컨트롤러: 중앙집중 네트워크 제어, 정책 기반 자동화
- NFV: 방화벽, 라우터 등 네트워크 기능을 가상 머신으로 구현
- 장점: 유연성, 자동화, 비용 절감, 빠른 서비스 배포
- 사용 사례: 데이터센터 네트워크, 5G 코어, SD-WAN
- 주요 기술: OpenvSwitch, ONOS, OpenDaylight, ETSI NFV

## 최신 트렌드
- Intent-Based Networking
- Cloud-Native NFV
