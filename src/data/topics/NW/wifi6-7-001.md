# 정의
WiFi 6의 6GHz 확장판인 WiFi 6E와, 초고속 멀티링크를 지원하는 차세대 WiFi 7 무선랜 표준입니다.

## 특징
- WiFi 6 (802.11ax): OFDMA, MU-MIMO, TWT, 최대 9.6Gbps
- WiFi 6E: WiFi 6 + 6GHz 대역 (간섭 적음, 채널 많음)
- 주파수: 2.4GHz, 5GHz, 6GHz (6E부터)
- OFDMA: 주파수 자원을 여러 기기에 분할 할당
- MU-MIMO: 다중 기기 동시 통신
- WiFi 7 (802.11be): Multi-Link Operation, 최대 46Gbps
- MLO: 2.4/5/6GHz 동시 사용으로 안정성 및 속도 향상
- 사용 사례: 고밀도 환경, AR/VR, 8K 스트리밍

## 최신 트렌드
- WiFi 7 상용화
- Multi-Link Operation
