# 정의
경영 전략은 조직의 장기적 목표 달성을 위해 자원을 배분하고 경쟁 우위를 확보하는 계획으로, 환경 분석과 전략적 의사결정을 포함합니다.

## 특징
- SWOT 분석: 내부 환경(강점Strengths, 약점Weaknesses)과 외부 환경(기회Opportunities, 위협Threats)을 분석하여 SO전략(강점-기회), WO전략(약점-기회), ST전략(강점-위협), WT전략(약점-위협) 도출
- PEST 분석: 거시 환경 분석 프레임워크로 정치적(Political), 경제적(Economic), 사회문화적(Social), 기술적(Technological) 요인 분석. PESTLE(법률Legal, 환경Environmental 추가)로 확장 가능
- Porter의 5-Force: 산업 구조 분석 모델로 1) 기존 경쟁자 간 경쟁, 2) 신규 진입자 위협, 3) 대체재 위협, 4) 공급자 교섭력, 5) 구매자 교섭력 분석
- MECE/LISS: MECE(Mutually Exclusive Collectively Exhaustive)는 상호배타적이고 전체포괄적인 논리구조, LISS(Large Issue, Small Step)는 큰 문제를 작은 단계로 나누어 해결하는 접근법
