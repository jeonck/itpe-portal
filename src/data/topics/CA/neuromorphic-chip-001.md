# 정의
뉴로모픽 칩(Neuromorphic Chip)은 인간 뇌의 구조와 작동 방식을 모방하여 설계된 차세대 인공지능 반도체. 폰노이만 구조의 컴퓨팅 한계(CPU와 메모리 간의 병목 현상)를 극복하고, 저전력으로 효율적인 병렬 연산을 수행하며, 특히 스파이킹 신경망(SNN) 기반의 학습 및 추론에 최적화되어 있는 기술. 엣지 디바이스에서의 AI 구현과 같은 분야에서 높은 잠재력을 보유.

## 특징
- 뇌 모방 아키텍처:
    - 폰노이만 구조와의 차이: 폰노이만 구조는 CPU와 메모리가 분리되어 데이터 이동이 빈번하지만, 뉴로모픽 칩은 프로세싱과 메모리 기능을 통합하여 데이터 이동을 최소화합니다.
    - 뉴런 및 시냅스 모방: 칩 내부에 뉴런과 시냅스 구조를 직접 구현하여 뇌의 병렬 처리 방식을 모방합니다.
- 스파이킹 신경망 (SNN, Spiking Neural Network):
    - 특징: 기존 딥러닝에서 사용하는 인공 신경망(ANN)과 달리, 뉴런이 특정 역치 이상의 자극을 받았을 때만 스파이크(Spike) 신호를 발생시켜 정보를 전달합니다.
    - 이벤트 기반 처리: 불필요한 연산을 줄여 극도로 낮은 전력 소비로 빠른 반응 속도를 가집니다.
    - 생물학적 현실성: 인간 뇌의 정보 처리 방식과 유사.
- 주요 장점:
    - 초저전력: 데이터를 이동시키지 않고 메모리에서 직접 연산(인메모리 컴퓨팅)하여 에너지 효율이 매우 높습니다.
    - 고효율 병렬 처리: 뇌의 뉴런들이 동시에 작동하듯이, 대규모 병렬 연산에 강합니다.
    - 지속적인 학습: 실시간으로 데이터를 학습하고 적응하는 능력을 가집니다.
- 주요 연구 개발 사례:
    - IBM TrueNorth: 4096개의 코어, 100만 개의 뉴런, 2억 5천만 개의 시냅스를 모방.
    - Intel Loihi: 비동기 스파이킹 뉴런과 온칩 학습 기능을 통합.
- 활용 분야: 엣지 디바이스에서의 AI(사물 인터넷, 로봇, 자율주행차), 센서 데이터 처리, 패턴 인식, 실시간 학습 등.
- 도전 과제: 기존 AI 모델 및 프레임워크와의 호환성, 프로그래밍 모델의 복잡성, 범용적인 적용의 어려움.

## 동작원리
뉴로모픽 칩은 뇌의 뉴런과 시냅스를 모방하여 동작합니다:

스파이킹 뉴런 동작 (Leaky Integrate-and-Fire 모델)
1) 입력 신호 수신: 시냅스를 통해 스파이크 신호 받음
2) 막전위 누적: 뉴런 내부의 막전위(Membrane Potential)가 상승
3) 누수(Leak): 시간이 지나면 막전위가 자연스럽게 감소
4) 발화(Fire): 막전위가 임계값(Threshold) 초과 시 스파이크 발생
5) 리셋: 막전위를 초기 상태로 리셋

시냅스 가소성 (STDP: Spike-Timing-Dependent Plasticity)
- 학습 메커니즘: 뉴런 간 연결 강도(Weight) 조정
- Pre-Synaptic 뉴런이 Post-Synaptic 뉴런보다 먼저 발화 → 시냅스 강화
- Post-Synaptic 뉴런이 먼저 발화 → 시냅스 약화
- 생물학적 학습 방식 모방

이벤트 기반 처리 (Event-Driven Processing)
- 스파이크가 발생할 때만 연산 수행
- 입력이 없으면 전력 소비 거의 없음
- 비동기 통신: 클럭 없이 이벤트 발생 시에만 동작
- 저전력 실시간 처리 구현
