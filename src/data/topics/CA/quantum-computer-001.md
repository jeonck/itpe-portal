# 정의
양자 컴퓨터는 양자 역학의 특성(중첩, 얽힘)을 활용하여 기존의 고전 컴퓨터가 해결하기 어려운 특정 문제들을 압도적으로 빠르게 풀 수 있는 차세대 컴퓨팅 기술입니다. 정보의 최소 단위인 큐비트(Qubit)를 사용하여 0과 1의 상태를 동시에 표현하고 상호작용함으로써, 기존 컴퓨터의 한계를 넘어서는 연산 능력을 제공합니다.

## 특징
- 큐비트 (Qubit):
    - 양자 컴퓨터의 최소 정보 단위.
    - 고전 컴퓨터의 비트가 0 또는 1 중 하나의 상태만 가질 수 있는 반면, 큐비트는 중첩(Superposition) 상태를 통해 0과 1의 상태를 동시에 가질 수 있습니다.
    - `n`개의 큐비트는 `2^n`개의 상태를 동시에 표현할 수 있어, 큐비트 수가 늘어날수록 연산 능력이 기하급수적으로 증가합니다.
- 중첩 (Superposition):
    - 큐비트가 여러 가능한 상태의 조합으로 동시에 존재할 수 있는 양자 역학적 특성. (예: 0이면서 동시에 1인 상태)
    - 이를 통해 고전 컴퓨터가 한 번에 하나의 상태만 처리하는 것과 달리, 양자 컴퓨터는 여러 상태를 동시에 탐색하고 연산할 수 있습니다.
- 얽힘 (Entanglement):
    - 두 개 이상의 큐비트가 서로 비국소적으로 연결되어, 하나의 큐비트 상태가 결정되면 다른 큐비트의 상태도 즉시 결정되는 양자 역학적 특성.
    - 얽힌 큐비트 간에는 거리에 상관없이 정보가 즉시 공유되는 것처럼 보이며, 이를 통해 양자 컴퓨터는 강력한 병렬 처리 능력을 얻습니다.
- 양자 알고리즘:
    - 쇼어(Shor) 알고리즘: 대규모 정수의 소인수분해를 효율적으로 수행하여 현재의 공개키 암호(RSA 등)를 무력화할 수 있습니다. (PQC 개발의 주요 동기)
    - 그로버(Grover) 알고리즘: 정렬되지 않은 데이터베이스에서 특정 항목을 O(루트N) 시간 복잡도로 탐색할 수 있습니다.
- 양자 우위 (Quantum Supremacy): 양자 컴퓨터가 고전 컴퓨터로 불가능하거나 실현 불가능한 시간 내에 특정 연산을 수행하는 능력.
- 구현 방식: 초전도 회로, 이온 트랩, 토폴로지컬 큐비트 등 다양한 물리적 방식으로 큐비트를 구현하고 제어하는 기술이 연구 중입니다.
- 과제: 큐비트의 안정성 및 오류율, 큐비트 확장성, 작동 환경(극저온) 등 기술적 난제.
