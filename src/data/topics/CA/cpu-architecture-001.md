# 정의
중앙처리장치의 내부 구조와 동작 원리로, 명령어 처리 효율을 높이기 위한 기법들을 포함합니다.

## 특징
- Pipeline: 명령어를 단계별로 동시 처리하여 성능 향상
- Cache: 자주 사용되는 데이터를 고속 메모리에 저장
- Branch Prediction: 분기 예측으로 파이프라인 효율 향상
- Hazard 해결: 데이터/제어/구조적 해저드 처리
