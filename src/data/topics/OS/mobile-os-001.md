# 정의
스마트폰과 태블릿을 위해 최적화된 운영체제로, 터치 인터페이스, 전력 관리, 모바일 앱 생태계를 지원합니다.

## 특징
- Android: Linux 커널 기반, 오픈소스 (AOSP), Google Play
- Android 구조: Linux Kernel → HAL → ART/Native Libraries → Framework → Apps
- ART (Android Runtime): Dalvik 후속, AOT 컴파일, JIT 병행, 성능 향상
- Dalvik: 과거 Android VM, DEX 파일, JIT 컴파일
- iOS: XNU 커널 (Mach + BSD), 폐쇄 생태계, App Store
- iOS 특징: Objective-C/Swift, Metal API, 강력한 보안 (Secure Enclave)
- 모바일 OS 특징: 터치 UI, 센서 통합, 배터리 관리, 샌드박스, 백그라운드 제한
- 기타 OS: HarmonyOS (Huawei), Fuchsia (Google, Zircon 커널)
