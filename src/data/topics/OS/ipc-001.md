# 정의
독립적인 프로세스들이 데이터를 교환하고 동작을 조정하기 위한 통신 메커니즘입니다.

## 특징
- Pipe (파이프): 단방향 통신, 부모-자식 프로세스, Anonymous/Named Pipe (FIFO)
- Message Queue: 메시지 단위 송수신, FIFO, 비동기, POSIX/System V
- Shared Memory (공유 메모리): 가장 빠른 IPC, 동일 메모리 영역 공유, 동기화 필요
- Semaphore: 프로세스 간 동기화, 카운터 기반, System V/POSIX
- Socket: 네트워크 기반 IPC, TCP/UDP, 프로세스 간/머신 간 통신
- Signal: 이벤트 통지, 비동기, SIGKILL, SIGTERM, SIGUSR1 등
- Memory-Mapped File: 파일을 메모리에 매핑하여 공유
- 선택 기준: 속도(공유메모리 > 메시지큐 > 파이프), 용도, 동기화 필요성
